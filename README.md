# Simple USA VPN

A simple, lightweight VPN setup designed to seamlessly route your connection through servers located near the United States. 

## 🛠 Maintenance & Setup

This VPN utilizes the free services provided by [VPNBook](https://www.vpnbook.com/). To maintain a stable connection, they regularly rotate their access passwords. You will need to periodically update your configuration to keep it working:

1. Visit [vpnbook.com](https://www.vpnbook.com/)
2. Navigate to the VPN server section on their website.
3. Locate the current password for your chosen server (usually displayed as an image).
4. Update your local VPN configuration with the newly provided password.
